# PEX---FERRAMENTA-DE-SIMULA-O-DE-ATENDIMENTO-COM-IA-
O PEX propõe uma ferramenta de simulação de atendimento com IA para treinar operadores em cenários realistas via chat ou voz. A solução reduz a carga dos instrutores, amplia a prática dos alunos e gera avaliações automáticas com feedback imediato e logs de desempenho.
